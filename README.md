# germantessmer.github.io

Personal academic website for Germán A. Tessmer, built with [Quarto](https://quarto.org) and deployed via GitHub Pages.

🌐 **Live site:** https://germantessmer.github.io/

---

## Stack

- **Framework:** Quarto 1.6.33
- **Language:** R 4.5.2
- **Deploy:** GitHub Pages → `main` branch → `/docs` folder

## Site structure

| File | Content |
|------|---------|
| `index.qmd` | About / bio |
| `research.qmd` | Working papers, data, publications |
| `teaching.qmd` | Teaching history |
| `cv.qmd` | Curriculum vitae |
| `styles.css` | Custom styles |
| `profile.jpg` | Profile photo |

## Update workflow

```bash
# 1. In RStudio: Build → Render Website

# 2. In Git Bash
cd /c/germantessmer.github.io
git add .
git commit -m "describe change"
git push
```

> **Always render before pushing.** GitHub Pages serves the `/docs` folder generated by Quarto.
